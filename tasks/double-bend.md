# double-bend
- category: loops

## Setting

```
|b |b |b |b |bD|
|k |k |k |kD|kW|
|k |k |kD|kD|k |
|k |kD|kD|k |k |
|kD|kD|k |k |k |
|kD|kD|k |k |k |
|k |kD|kD|k |k |
|k |k |kD|kD|k |
|k |k |k |kD|kW|
|k |k |k |k |kS|
```

- length: 5

## Solution

```
repeat 2:
    repeat 4:
        left()
    repeat 4:
        right()
```
